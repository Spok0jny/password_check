# Simple Password Strength Checker

A basic Python tool designed to evaluate password security. This was my first project while transitioning from C# to Python, created as part of my journey toward cybersecurity certification.

##  Features
- **Length Verification:** Checks if the password meets the minimum 8-character requirement.
- **Awards points for including:** 
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Special characters (using Python's `string.punctuation` library)
- **Interactive CLI:** Users can check multiple passwords in one session.

##  Requirements
- Python 3.x

### Learning Goals
This project helped me understand:
- Python syntax and indentation (vs C# curly braces).
- Functional programming in Python.
- Working with the standard string library.
